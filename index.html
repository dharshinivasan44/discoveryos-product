Help & Documentation

Getting Started

DiscoveryOS helps you analyze customer feedback to discover product insights.

  1. Upload customer feedback files
  2. DiscoveryOS processes and clusters the feedback
  3. View insights by different strategies
  4. Export findings for your product roadmap

Supported File Formats

  • CSV - Customer feedback in tabular format
  • JSON - Structured feedback data
  • TXT/MD - Plain text or markdown
  • VTT/SRT - Transcribed interviews
  • XLSX/XLS - Excel spreadsheets
